# Cardano Ogmios Examples
A Yarn Workspace containing examples written in TypeScript using the `@cardano-ogmios/client`
package:

- :construction: [@cardano-ogmios-examples/payment-notifier](./packages/payment-notifier/README.md) 

## Development
#### Install
```console
yarn install
```
### Run the docker-compose stack
#### mainnet
```console
yarn mainnet:up
```
#### testnet
```console
yarn testnet:up
```
### Stop the docker-compose stack
#### mainnet
```console
yarn mainnet:down
```
#### testnet
```console
yarn testnet:down
```
### Lint
```console
yarn lint
```
### Cleanup
```
yarn cleanup
```
